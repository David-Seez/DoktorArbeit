By assumption for each $T_i$ there is a $\tilde{x}_i\in V_j$ and a $\lambda_i$ such that $\lambda_i \tilde{x}_i \nin V_j$. Define $x_i\coloneq \lambda_i\tilde{x}_i$. Then if $i$ approaches infinity, $\delta_i$ approaches $0$ and hence $x_i$ approaches $x_j$, whilst never reaching it. Said differently: there are infinitly many different $x_i$. All those come with different flow lines $\gamma_i$ between different critical points. 
\end{lemma}
